# Smart Plug Alliance 🔌
A sustainable, voice-controlled smart plug built with ESP32.
Supports Alexa & Siri via Matter protocol.

## Tech Stack
- Hardware: ESP32, 16A Relay, PZEM-004T
- Firmware: ESPHome
- Hub: Home Assistant
- Protocol: Matter
